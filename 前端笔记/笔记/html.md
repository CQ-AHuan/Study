# HTML 5

> 一堆标签

## 朴素标签 5

```html
<!-- 最朴素的块级元素 默认样式: display: block; -->
<div></div>

<!-- 最朴素元素 无默认样式 -->
<span></span>
```

## 文字标签 3

```html
<!-- 段落标签 块级 -->
<p></p>

<!-- 标题标签 块级 -->
<h1></h1>
...
<h6></h6>

<!-- 斜体标签 行级 -->
<i>hello world!</i>

<!-- 加粗标签 行级 -->
<strong>hello world!</strong>
<b>hello world!</b>
```

## 列表标签 5

```html
<!-- 无序列表 unordered list 块级 -->
<ul>
    <!-- 列表项 list item 块级 -->
    <li></li>
</ul>

<!-- 有序列表 ordered list 块级 -->
<ol>
    <!-- 列表项 list item 块级 -->
    <li></li>
</ol>
```

## 表格标签  2

```html
<!-- 表格套餐都是特殊的块级元素 -->
<table>
    <!-- 表头 -->
    <thead>
    	<tr>
        	<th>表头1</th>
        	<th>表头2</th>
        	<th>表头3</th>
        </tr>
    </thead>
    <!-- 表身体 -->
    <tbody>
    	<tr>
        	<td>单元格1</td>
        	<td>单元格2</td>
        	<td>单元格3</td>
        </tr>
    	<tr>
        	<td>单元格1</td>
        	<td>单元格2</td>
        	<td>单元格3</td>
        </tr>
    </tbody>
</table>
```

## 表单标签 5

> 表单：用户录入信息用的东西

```html
<!-- 输入框 行级块 -->
<input type="text">

<!-- 密码输入框 行级块 -->
<input type="password">

<!-- 复选框 -->
<input type="checkbox">

<!-- 单选框 -->
<input type="radio">

<!-- 下拉选择框 -->
<select>
    <option>1</option>
    <option>2</option>
    <option>3</option>
</select>

<!-- 段落文本输入框 -->
<textarea></textarea>

<!-- 按钮 -->
<button>点击</button>
```

## 音视频 3

```html
<!-- 音频 -->
<audio></audio>

<!-- 视频 -->
<video></video>
```
