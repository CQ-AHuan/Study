# 命令行基础 5

## 通用命令

```shell
# 打印当前目录
pwd

# 进入到某个文件夹
cd "文件夹目录"
# 回到根目录
cd ~ 
# 回到上级目录
cd ..

# 查看当前文件里中的东西
ls

# 清空
# command + r (刷新) (mac)
clear

# 取消当前动作
快捷键: ctrl + c

# 输入上一条命令
快捷键: 向上箭头
```
