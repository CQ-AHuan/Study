# 判断 5

## if / else / elseif 5

### 语法

```js
// if
// 单条件判断
if (条件表达式) {
    // 条件为真时要运行的代码
}

// if ... else
// 如果...否则...
if (条件表达式) {
    // 条件为真时要运行的代码
} else {
    // 否则要运行的代码
}

// if ... elseif ... else
// 多路判断
if (条件表达式1) {
    // 条件表达式1为真时要运行的代码
} else if (条件表达式2) {
    // 条件表达式2为真时要运行的代码
} else {
    // 否则要运行的代码
    // (既不满足表达式1又不满足表达式2)
}
```

## switch 5

> 多路全等

```js
switch (要被判断的值/表达式) {
    case 值1:
        // 要被判断的值/表达式 === 值1 的时候 要执行的代码
        break; // 打断
    case 值2:
        // 要被判断的值/表达式 === 值2 的时候 要执行的代码
        break; // 打断
    case 值3:
        // 要被判断的值/表达式 === 值3 的时候 要执行的代码
        break; // 打断
    default:
        // 以上条件都不满足时 要执行的代码
        break;
}
```
